Sobre o Projeto

Olá!
Aqui iremos compreender como funciona nosso projeto.
Nossa página foi escrita em HTML, CSS e JavaScript, fazendo referência ao personagem Sheldon Cooper da série Big Bang Theory.
Nessa série, o personagem interpretado por Jim Parson é um gênio da física, que com seus amigos, protagonizam o sitcom de grande sucesso.

Vamos lá! ou melhor:


Abaixo poderemos analizar cada etapa da criação da página.
  1. A estrutura básica da página:
    • Html - Define o início de um documento HTML.
    • head - Contém informações sobre o documento, como título e metadados.
    • meta charset - Define o conjunto de caracteres universal UTF-8 para a página.
    • meta name - Configura a viewport para dispositivos móveis.
    • title - Define o título da página exibido na aba do navegador.
    • link rel - Importa uma folha de estilo externa chamada "styles.css".
    • style - Permite a inclusão de estilos CSS diretamente no documento HTML.
    • body - Contém o conteúdo visível da página.

Nessa área temos o CSS ditando a estilização da página.
  1. A estrutura básica do CSS:
    • neon-text - Uma classe CSS que estiliza o texto com efeito gradiente neon e sombra. Ela define o tamanho da fonte, a cor do texto como transparente (para que o gradiente apareça), o uso de background-clip para aplicar o gradiente ao texto e animações para criar os efeitos de gradiente e sombra.
    • code-container -
    • code-box -
    • code-image -
    • code-scroll -
    • Os códigos acima são classes CSS usadas para estilizar as caixas de código com imagens. Elas controlam o layout, a aparência e a posição das imagens dentro das caixas de código.
    • Keyframes neonAnimation - Define uma animação chamada "neonAnimation" que cria o efeito de gradiente neon. Ele altera a posição de fundo do gradiente de 0% para 100%, criando uma animação de movimento de gradiente.
    • Keyframes shadowAnimation - Define uma animação chamada "shadowAnimation" que cria o efeito de sombra intermitente no texto. Ela altera a propriedade text-shadow para criar a animação de piscar.

Aqui, sequencialmente, temos o JavaScript!
  1. A estrutura básica do JavaScript:
    • Seleção de elementos DOM
      • Neste trecho de código abaixo, o código está selecionando elementos do DOM (Document Object Model) usando `document.getElementById()` e armazenando-os em variáveis para uso posterior. Os elementos selecionados são:
        • searchForm - O elemento com o ID "search-form", que é o formulário de pesquisa.
        • searchInput - O elemento com o ID "search-input", que é a caixa de entrada de pesquisa.
        • searchResults - O elemento com o ID "search-results", que é a lista de resultados de pesquisa.
        • clearButton - O elemento com o ID "clear-button", que é o botão para limpar a pesquisa.
  2. Evento de envio de Formulário
    • Este trecho de código abaixo adiciona um ouvinte de evento ao formulário de pesquisa (elemento searchForm) para o evento "submit". Quando o formulário é enviado, a função callback é chamada.
    • `e.preventDefault()` - Isso impede o comportamento padrão de envio do formulário, que faria com que a página fosse recarregada.
    • `performSearch()` - Chama a função `performSearch()` para executar a pesquisa.
  3. Evento de Tecla Pressionada na Caixa de Entrada
    • Este trecho de código abaixo adiciona um ouvinte de evento à caixa de entrada de pesquisa (elemento searchInput) para o evento "keydown" (quando uma tecla é pressionada).
    • Ele verifica se a tecla pressionada é a tecla "Enter" (código de tecla "Enter" é igual a "Enter").
    • Se a tecla "Enter" for pressionada, a função callback é chamada.
    • `e.preventDefault()` - Novamente, isso impede o comportamento padrão de submissão do formulário.
    • `performSearch()` - Chama a função `performSearch()` para executar a pesquisa.
  4. Evento de Clique no Botão de Limpar
    • Este trecho abaixo de código adiciona um ouvinte de evento ao botão de limpar (elemento clearButton) para o evento "click" (quando o botão é clicado).
    • Quando o botão é clicado, a função callback é chamada.
    • `searchInput.value = "";` - Isso limpa o conteúdo da caixa de entrada de pesquisa, definindo seu valor como uma string vazia.
    • `hideSearchResults();` - Chama a função `hideSearchResults()` para ocultar a lista de resultados de pesquisa.
  5. Função `performSearch()`
    • A função `performSearch()` é responsável por executar a pesquisa.
